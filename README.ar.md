# جـتك (Gtk)
[[English]](README.md)

روابط GTK+ للغة البرمجة الأسس، توفر الوصول إلى مجموعة أدوات GTK 3 لبناء واجهات المستخدم الرسومية.

## الوصف

توفر هذه المكتبة روابط لغة الأسس لـ GTK+ 3، مما يتيح للمطورين إنشاء تطبيقات سطح مكتب حديثة مع واجهات مستخدم رسومية باستخدام لغة البرمجة الأسس. تغلف الروابط مكتبة GTK+ المكتوبة بلغة C وتعرض وظائفها من خلال واجهة متوافقة مع الأسس.

## المكونات

- **جـتك (Gtk)**: الروابط الأساسية لـ GTK+ 3 وتشمل:
  - `مشترك.أسس` (`common.alusus`) - أنواع ودوال GTK الشائعة
  - `ثوابت.أسس` (`constants.alusus`) - ثوابت GTK والتعدادات
  - `ودجات.أسس` (`widgets.alusus`) - روابط الودجات (الأزرار، التسميات، حقول الإدخال، إلخ)
  - `نوافذ.أسس` (`windows.alusus`) - إدارة النوافذ والحاويات

- **جـلب (Glib)**: روابط مكتبة GLib وتوفر:
  - `مشترك.أسس` (`common.alusus`) - أنواع وأدوات GLib الشائعة
  - `قـائمة.أسس` (`List.alusus`) - هياكل بيانات قوائم GLib

- **Examples**: تطبيقات نموذجية توضح استخدام المكتبة:
  - `hello_world.alusus` - مثال أساسي لنافذة GTK
  - `simple_timer.alusus` - تطبيق مؤقت
  - `widgets_guide.alusus` - أمثلة على استخدام الودجات
  - `factorial.alusus` - تطبيق تجريبي لحساب المضروب
  - `بسملة.أسس` - مثال بسيط لنافذة ج.ت.ك
  - `دليل_الودجات.أسس` - أمثلة على استخدام الودجات
  - `مضروب.أسس` - تطبيق بسيط لحساب المضروب

## الاستخدام

اشمل المكتبة في برنامج الأسس الخاص بك:

```

مـحا.اشمل_ملف("Alusus/Gtk"، "جـتك.أسس")؛
```
